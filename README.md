## sogou-ocr [![sogou-ocr](https://img.shields.io/npm/v/sogou-ocr.svg)](https://npmjs.org/sogou-ocr)

> 

### Installation

```bash
$ npm install sogou-ocr
```

### Example

```js
const SogouOCR = require('sogou-ocr');

SogouOCR.recognize('~/Desktop/demo.png')
.then(text => {
  console.log(text);
});
```
